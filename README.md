## Important steps to run the app

- The project can be imported to the IDE of your choice (preferably Eclipse and IntelliJ, which have been 100% tested), as an existing Maven project.
- The app uses MySQL, so it is necessary to use a compatable databse.
- The database of the app can be created either by the sql scripts that can be found in the folders sql_scripts
- Change the application properties file in the app to **include the username and password of <ins>your</ins> sql connection**.
- Run the app by right clicking the TraineeshipappApplication.java file in the \src\main\java\myy803\traineeshipapp directory, and selecting to **run it as a Java application**.
- Type on your browser's search bar http://localhost:8080/ to access the app.
